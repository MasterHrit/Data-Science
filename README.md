## How to Run

### Clone Repository

```bash
git clone https://github.com/MasterHrit/Python-and-Data-Science.git
```

### Navigate to Repository

```bash
cd Python-and-Data-Science
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Jupyter Notebook

```bash
jupyter notebook
```
